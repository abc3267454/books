Readme of e2eSoft FlvCapture
============================


FlvCapture is a software which can monitor all the FLV/MP4 videos in any video website (independent of the website design), and get the URL of the video, so you can download or watch it easily. 

It can detect the video file saved in internet temporary folder as well, so you can get the video directly if you already visited it.


[LIMITATION]
It only works on 32bits windows.
